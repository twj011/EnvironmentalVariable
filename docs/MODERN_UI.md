# 🚀 现代化环境变量管理器 Pro

## ✨ 新特性

### 🎨 现代化设计
- **深色主题** - 基于 Catppuccin Mocha 配色方案
- **浅色主题** - 一键切换（点击右上角 🌙 按钮）
- **卡片式布局** - 每个变量都是独立的卡片，清晰易读
- **渐变按钮** - 炫酷的渐变色按钮和悬停效果
- **圆角设计** - 所有元素都采用圆角设计，更加柔和
- **平滑动画** - 卡片入场动画，流畅的交互体验

### 📊 智能仪表板
左侧仪表板实时显示：
- 📦 总变量数量
- 👤 用户变量数量
- ⚙️ 系统变量数量
- 🛠️ 开发工具变量（自动识别 Java、Python、Node.js 等）
- 🎯 PATH 条目数量

### 🔍 智能分类
自动将变量分为三大类：
- **🛠️ 开发工具** - Java、Python、Node.js、Git、Maven、Gradle 等
- **⚙️ 系统路径** - PATH、TEMP、WINDIR、SYSTEMROOT 等
- **👤 用户配置** - 其他自定义变量

### 🎯 智能 PATH 编辑器
- 可视化列表展示所有 PATH 条目
- 拖拽排序调整优先级
- 蓝色高亮显示变量引用（如 `%JAVA_HOME%`）
- 红色标记无效路径
- 一键智能优化建议

### 🚀 快速操作
- **实时搜索** - 输入关键词即时过滤变量
- **分类筛选** - 按类别快速查看变量
- **一键编辑** - 每个卡片都有编辑和删除按钮
- **权限提示** - 清晰显示当前权限状态

## 🎮 使用方法

### 启动程序
```bash
# 用户模式（只能编辑用户变量）
uv run python main.py

# 管理员模式（可编辑系统变量）
# 右键以管理员身份运行终端，然后执行：
uv run python main.py
```

### 主要功能

#### 1. 查看变量
- 所有变量按类别自动分组显示
- 左侧仪表板显示统计信息
- 每个变量卡片显示名称、值和作用域（用户/系统）

#### 2. 搜索变量
- 在顶部搜索框输入关键词
- 实时过滤显示匹配的变量

#### 3. 编辑变量
- 点击变量卡片上的 "✏️ 编辑" 按钮
- 输入新值后保存

#### 4. 删除变量
- 点击变量卡片上的 "🗑️" 按钮
- 确认后删除

#### 5. 新建变量
- 点击顶部 "➕ 新建变量" 按钮
- 输入变量名和值

#### 6. 编辑 PATH
- 点击 "🎯 编辑 PATH" 按钮
- 在可视化编辑器中拖拽排序
- 添加或删除路径条目

#### 7. 切换主题
- 点击右上角 "🌙 深色" 按钮
- 在深色和浅色主题之间切换

## 🎨 设计亮点

### 配色方案
- **深色主题**：基于 Catppuccin Mocha
  - 背景：`#1e1e2e`
  - 卡片：`#313244`
  - 强调色：`#89b4fa`（蓝色）、`#a6e3a1`（绿色）、`#f38ba8`（红色）

- **浅色主题**：基于 Catppuccin Latte
  - 背景：`#eff1f5`
  - 卡片：`#ffffff`
  - 强调色：`#1e66f5`（蓝色）

### 交互设计
- 悬停效果：所有可点击元素都有悬停反馈
- 动画效果：卡片入场动画（300ms 缓动）
- 视觉反馈：操作成功/失败都有明确提示

## 🔧 技术栈
- **PyQt6** - 现代化 GUI 框架
- **QSS** - 自定义样式表（类似 CSS）
- **QPropertyAnimation** - 平滑动画效果
- **Windows Registry API** - 环境变量操作
